Welcome to my learning journey repository! This repository contains my notes, code, and projects as I explore various topics in data science, machine learning, and programming.

## Repository Structure

### 📊 Data Science & Analytics
- **Pandas/**: Data manipulation and analysis with Pandas
  - `Day1.ipynb`: Introduction to Pandas DataFrames and basic operations
  - `Dataset/`: Sample datasets for practice
- **Numpy/**: Numerical computing with NumPy
- **Matplotlib/**: Data visualization with Matplotlib
- **Seaborn/**: Statistical data visualization

### 🤖 Machine Learning
- **Machine Learning/**: ML algorithms and implementations
  - `dataset/`: ML datasets

### 💻 Programming & Algorithms
- **DSA (Python)/**: Data Structures and Algorithms implementations in Python

## Getting Started

### Prerequisites
- Python 3.7+
- Jupyter Notebook/JupyterLab
- Required libraries: pandas, numpy, matplotlib, seaborn, scikit-learn

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/ahnaf-learning.git
cd ahnaf-learning

# Install dependencies (if requirements.txt exists)
pip install -r requirements.txt
```

## Usage

Each folder contains Jupyter notebooks with explanations and code examples. Start with the introductory notebooks in each section:

1. **Pandas**: Begin with `Day1.ipynb` for DataFrame basics
2. **Numpy**: Explore numerical operations
3. **Matplotlib/Seaborn**: Learn data visualization techniques
4. **Machine Learning**: Dive into ML algorithms
5. **DSA**: Practice algorithms and data structures
